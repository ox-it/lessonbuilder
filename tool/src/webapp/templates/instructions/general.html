STARTING OUT

We suggest that you start out by using Add Text to put material on this page.

Alternatively, you can use Add Multimedia for a Flash presentation, movie, etc.

Then you can put links to additional resources, assignments for the unit, etc, by using the other buttons. If you place the mouse over a button you'll get a brief description of what it does.

Once you've added content, you'll find Edit buttons next to each item. They can be used to adjust the size of multimedia objects, to change the title, to add descriptive text, etc. They can also activate features that you can use to control the sequence in which items are available.

Creating Multiple Pages

If you want to use multiple pages, there are two ways to do it.

  1. You can go to Edit Tools in Site Info and add multiple copies of Lesson Builder. Each copy will appear separately in the left margin. You might create an instance for each unit of your course.
  2. If you want several pages in one unit, you can create a main page for each unit, and then use Add Subpage several times. Once you've created the pages, click on each one to go to the page and add content.

Other documentation

  • The Sakai help system has additional details. Use the Help icon at the upper right of every Sakai tool.
  • The multimedia FAQ has specifics on including multimedia content in your page.
  • The accessibility introduction has a brief description of tools in Lesson Builder to help create accessible content, and pointer to some online information on Web Accessibility.
